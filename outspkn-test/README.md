# OUTSPKN - UK Car Culture Brand Ecommerce Website

A static HTML, CSS, and JavaScript ecommerce website for OUTSPKN, a UK-based car culture brand selling window stickers, general stickers, clothing, custom number plates, and custom laser cut car badges.

## Project Overview

This project is a fully responsive ecommerce website built with vanilla HTML, CSS, and JavaScript. It doesn't require any server-side processing or databases, making it easy to deploy on any web hosting service.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Product catalog with filtering by category
- Shopping cart functionality using localStorage
- Product detail pages
- About and Contact pages
- Newsletter subscription form
- Mobile-friendly navigation

## Pages

1. **Home Page (index.html)** - Main landing page with featured products and categories
2. **Shop Page (shop.html)** - Product catalog with filtering options
3. **Cart Page (cart.html)** - Shopping cart with product details and checkout option
4. **About Page (about.html)** - Information about the brand and team
5. **Contact Page (contact.html)** - Contact form and information

## Setup Instructions

1. Clone or download this repository
2. Create an `img` directory with the following subdirectories:
   - `products` - For product images
   - `team` - For team member photos
3. Add necessary images to the `img` directory
4. Open `index.html` in your web browser to view the website
